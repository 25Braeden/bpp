Unit 1: Variables

Variables in Python

In Python (and all other coding languages), variables are used to store information that can be accessed and modified throughout your program. They act as containers for data, such as numbers, text, or more complex structures. To create a variable in Python, you simply assign a value to a name using the equals sign (=). For example:

x = 5
name = "Alice"

In the example above, x is assigned the value 5, and name is assigned the text value "Alice". Python is dynamically typed, which means you don’t need to specify the type of data a variable holds.

Let's go over some things you can do with variables.

  • Add and subtract: 

      x = 5
      y = 2
      z = x + y -> z is equal to 7

  • Combine strings: 

      first_name = "Alice"
      last_name = "Smith"
      full_name = first_name + " " + last_name -> full_name is equal to "Alice Smith"
