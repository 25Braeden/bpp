Taking User Input

You can take user input in python by using the input() method.

  • Using in a print statement: 
    • print(input()) will output whatever the user types
  • Alternatively, you can store the input in a variable, we will get more into variables later: 
    • name = input()
      print(name) will also output whatever the user entered.
  • Additional information:
    • You can add text inside of the input() method to give the user context. For example if you run the code: 
      print(input(Enter your age: ))
      it will prompt the user to enter their age then output the result.
