Here's some things you need to know about print():

  • print() is a function that takes in a string and displays it to the console.
  • Strings are sequences of characters enclosed in quotation marks.
    • Strings are anything enclosed in (' ') or (" ").
  • Strings can be concatenated using the + operator.
    • For example, the code: print("Hello" + "World") will output "HelloWorld".
