Unit 0: Intro to Python

Your First Program: Hello World

Before we get into all the boring stuff like datatypes and logical operators, let's write our first program! In Python, the "Hello World" program is a simple one-liner that prints the phrase "Hello, World!" to the console. Here's how you can do this:

  1. Open python compiler of choice
    • You can use this online compiler for this project.
  2. Copy and paste the following code: 
    print("Hello, World!")
  3. Run the program!
