Unit 0: Intro to Python

Why Learn Python?

  • Easy to Understand.
    • Python is written in a way that's close to everyday English, so it's great for people who are new to programming.
  • Use it everywhere!
    • Python can help you make websites, analyze data, or even program robots and games!
  • Used in Real Jobs.
    • Learning Python is a great start for people who want to work with technology in the future, like making apps or analyzing data for companies.

Some companies that use Python:

  • Google
  • Facebook
  • Instagram
  • Spotify
  • Netflix
